# ai-voice-answers

A click-to-talk interface that turns voice questions into spoken AI answers.

![logo](https://raw.githubusercontent.com/trucomanx/AiVoiceAnswers/main/screenshot.png)

## 1. Installing

To install the package from [PyPI](https://pypi.org/project/ai_voice_answers/), follow the instructions below:


```bash
pip install --upgrade ai_voice_answers
```

Execute `which ai-voice-answers` to see where it was installed, probably in `/home/USERNAME/.local/bin/ai-voice-answers`.

### Using

To start, use the command below:

```bash
ai-voice-answers
```
