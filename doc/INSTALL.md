# ai-voice-answers

A click-to-talk interface that turns voice questions into spoken AI answers.


## Install from PYPI

The homepage in pipy is https://pypi.org/project/ai_voice_answers/

```bash
pip install --upgrade ai_voice_answers
```

Using:

```bash
ai-voice-answers
```

## Install from source
Installing `ai-voice-answers` program

```bash
git clone https://github.com/trucomanx/AiVoiceAnswers.git
cd AiVoiceAnswers
pip install -r requirements.txt
cd src
python -m build
pip install dist/ai_voice_answers-*.tar.gz
```
Using:

```bash
ai-voice-answers
```

## Uninstall

```bash
pip uninstall ai_voice_answers
```
